## Small and Medium Hospital Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/26e1cf30bc204291bef092a3fa5d4172/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Outpatient Information
5. Pharmacy Information
6. System Announcements
7. Personal Center
8. My Favorites
9. Backend Management


**Administrator Functions:**
1. Login
2. Personal Center
3. Change Password
4. Personal Information
5. User Management
6. Doctor Management
7. About Us Management
8. Department Type Management
9. Outpatient Information Management
10. Pharmacy Information Management
11. Payment List Management
12. Drug Type Management
13. System Management (System Announcements, Carousel Management)
